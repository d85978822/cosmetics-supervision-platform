# 各子系统数据流转图设计

## 1. 化妆品注册人备案人信息档案系统数据流转

### 1.1 主数据流
```mermaid
graph TD
    A[企业] -->|提交备案资料| B[备案受理]
    B -->|基础信息| C[信息档案库]
    B -->|信用信息| D[信用评价]
    D -->|评价结果| C
    C -->|档案信息| E[数据中台]
    E -->|标准数据| F[其他业务系统]
```

### 1.2 信息归集流
```mermaid
graph TD
    A[许可系统] -->|许可信息| B[信息归集]
    C[监管系统] -->|监管信息| B
    D[风险系统] -->|风险信息| B
    B -->|整合信息| E[档案库]
    E -->|信用数据| F[信用评价]
    F -->|评级结果| E
```
